üì± Test Cases ‚Äî iOS Session Persistence

TC-001 CRITICAL
Initial Login Success

Steps

  1. Open Safari on iPhone
  2. Navigate to https://super-flashcards.rentyourcio.com
  3. Tap "Sign in with Google"
  4. Complete Google OAuth flow

Expected Result

Login completes successfully. User sees their dashboard/home screen with their data.

TC-002 CRITICAL
Session Survives App Switch (Short)

Steps

  1. While logged in, press Home button (or swipe up)
  2. Open another app (e.g., Messages, Settings)
  3. Wait 30 seconds
  4. Return to Safari

Expected Result

User is STILL logged in. No login prompt. Dashboard shows immediately.

TC-003 CRITICAL
Session Survives App Close + Reopen

Steps

  1. While logged in, swipe up from bottom to show app switcher
  2. Swipe Safari UP to force close it completely
  3. Wait 10 seconds
  4. Open Safari again
  5. Navigate to https://super-flashcards.rentyourcio.com

Expected Result

User is STILL logged in. No login prompt. Session persisted across app restart.

TC-004 HIGH
Session Survives Extended Idle (5 minutes)

Steps

  1. While logged in, lock the iPhone (press side button)
  2. Set a timer for 5 minutes
  3. After 5 minutes, unlock iPhone
  4. Return to Safari / Super-Flashcards tab

Expected Result

User is STILL logged in. Page may refresh but should auto-restore session.

TC-005 HIGH
Login Prompt Speed (If Logged Out)

Steps

  1. Clear Safari cookies for the site (Settings > Safari > Clear History)
  2. Open Safari and navigate to https://super-flashcards.rentyourcio.com
  3. Time how long until login button/prompt appears

Expected Result

Login prompt appears within 3-5 seconds. NOT 30+ seconds like before.

TC-006 MEDIUM
Comparison with MetaPM (Control)

Steps

  1. Open Safari on iPhone
  2. Navigate to https://metapm.rentyourcio.com
  3. Login to MetaPM
  4. Force close Safari, reopen, navigate back
  5. Compare behavior with Super-Flashcards

Expected Result

Super-Flashcards should behave the SAME as MetaPM ‚Äî both stay logged in.
